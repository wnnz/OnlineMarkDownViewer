# 欢迎使用 MD Viewer

这是系统自带的默认文档源首页，用来帮助你快速确认在线 Markdown 查看器是否已经正常工作。

## 你可以先体验这些功能

- 切换左侧目录
- 使用搜索框按文件名和内容搜索
- 打开不同文档观察右侧章节导航
- 测试代码高亮、表格、任务列表和引用块样式

## 默认账号

- 用户名：`admin`
- 密码：`admin123`

## 目录建议

先阅读：

1. `01-quick-start/usage-guide.md`
2. `02-configuration/source-config-guide.md`
3. `03-markdown-examples/markdown-demo.md`

读完这几篇之后，再接入你自己的实际文档目录会更顺手。
